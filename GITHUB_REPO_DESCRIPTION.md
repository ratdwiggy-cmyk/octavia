## Longer description (for the README badge area or a social preview card, optional)

```
Octavia is a personal-site template for people who build things in public.
It ships as a homepage, a Collaborate page, and a six-section "Operating
System" that indexes every project by pipeline stage — Problem, Research,
Prototype, Test, Improvement, Result — so visitors can see not just what
you built, but how you got there. No framework, no build step, no
dependencies: open index.html and it already works. Fill in the
placeholders, push to GitHub Pages, and it's live.
```
